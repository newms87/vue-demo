
# vue-demo (alpha)

### Clone the repo
```
git clone https://github.com/basandlin/vue-demo.git
```
### Install dependencies
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Navigate to localhost:8080


### *For Future*
### Run Tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
